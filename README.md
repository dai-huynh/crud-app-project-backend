# server-code

**This is a project for Hunter College: CSCI 39548.**
This repository is the server (back-end) code for Final Project - Full-Stack CRUD Application.

### Tech Stack

- React Router
- React Redux
- Node
- Express
- PostgreSQL
- Sequelize

### Get Started

In database/utils/configDB.js:

```
const dbPwd = "<YOUR PASSWORD>";
```

In the terminal, run:

```
npm install
npm start
```
